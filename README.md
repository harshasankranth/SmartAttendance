# SmartAttendance
AI-powered facial recognition attendance system built with DeepFace, FastAPI, and React.

## Tech Stack
- Face Recognition: DeepFace + OpenCV
- Backend: FastAPI + WebSockets
- Frontend: React + Tailwind CSS
- Database: Supabase PostgreSQL
